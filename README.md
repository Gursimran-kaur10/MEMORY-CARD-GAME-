
MEMORY CARD GAME :
===================

This project is a text-based game where:
Cards are placed face-down
The player selects two card positions
If both match, they remain revealed
Otherwise, they hide again
Game ends when all pairs are matched

------------------------------------

The program uses:
✔ Stack to store each card selection
✔ Random shuffle to generate a new board every time
✔ Arrays to track card values and visibility

-------------------------------------
FEATURES :

Random shuffled card layout
Stack-based selection storage
Simple and interactive text UI
Supports up to 10 pairs

-------------------------------------
Tracks:
Total attempts
Stack size
Matched pairs

-----------------------------------

Technologies Used:

C++
Stack Data Structure
Arrays
Randomization (rand, srand)
GitHub Codespaces / Local Compiler

------------------------------------
