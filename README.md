
MEMORY CARD GAME :
===================
RUN COMMANDS(IN CODESPACE) TO START GAME :
1. chmod +x run.sh
2. ./run.sh

-------------------------
This project is a text-based game where:
1. Cards are placed face-down
2. The player selects two card positions
3. If both match, they remain revealed
4. Otherwise, they hide again
5. Game ends when all pairs are matched

------------------------------------

The program uses:
✔ Stack to store each card selection
✔ Random shuffle to generate a new board every time
✔ Arrays to track card values and visibility

-------------------------------------
FEATURES :

✔ Random shuffled card layout
✔ Stack-based selection storage
✔ Simple and interactive text UI
✔ Supports up to 10 pairs

-------------------------------------
Tracks:
✔ Total attempts
✔ Stack size
✔ Matched pairs

-----------------------------------

Technologies Used:

1. C++
2. Stack Data Structure
3. Arrays
4. Randomization (rand, srand)
5. GitHub Codespaces / Local Compiler
